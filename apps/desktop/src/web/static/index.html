朋友制作器

专为 Switch 版《朋友收集：梦想生活》制作

现在这版 Web UI 已经覆盖三块核心工作：生成绘图脚本、刷入 ESP32 固件，以及进行手柄动作测试，面向 Switch 版《朋友收集：梦想生活》的自动绘制流程持续完善中。

Current Mode Studio + Flash + Test

输入

导入图片，设定单色或官方色绘制参数，然后发送到 ESP32 开始绘制。

需要测试 需要先进行手柄测试

当前还没有确认开发板已经连上 Switch。开始绘制前，请先到“手柄测试”页完成连接。

绘制模式

当前固定走 250x250 画布并从画布中心起步。单色模式画黑白图，官方色模式会量化到游戏内置的 7x12 色盘；画笔大小按下方选择生成。

深色像素会绘制，浅色像素会保留为空白背景，不需要切换调色板。
