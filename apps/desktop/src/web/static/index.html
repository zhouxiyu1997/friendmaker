朋友制作器 / Friend Maker

自动绘画的基本原理，是通过数据线连接电脑与 ESP32 开发板，先刷入固定固件，再由电脑发送绘图指令，开发板按规律模拟 Switch 手柄输入完成自动绘画。

推荐顺序是先刷固件，再做手柄测试，最后回到“脚本生成”页导入图片并先点“仅生成命令”。手柄测试未通过时，不允许开始正式绘制。开始前请确认 Switch 处于初始绘画页面、画笔停在画布中心、侧边色彩槽保持默认；彩色模式请使用“官方色绘制”，自定义调色板暂未开放。建议首次先用简单图和大画笔测试，开始绘制后不要再操作手柄或触碰屏幕，以免错位。

Current Mode Studio + Flash + Test
推荐流程
  1. 先刷固件 先到“刷入固件”更新 ESP32，确保后面的蓝牙连接和按键测试基于最新版本进行。
  2. 再测连接 进入“手柄测试”确认开发板已经能被 Switch 正常识别、连接，并进入可发送状态。
  3. 最后导图 确认连接没问题后，再回到“脚本生成”输入图片、调整参数并开始正式绘制。

绘制模式

当前固定走 256x256 脚本坐标画布并从画布中心起步。单色模式画黑白图，官方色模式会量化到游戏内置的 7x12 色盘；画笔大小按下方选择生成。建议在游戏里切换为方块笔刷，整体观感通常会更美观。

深色像素会绘制，浅色像素会保留为空白背景，不需要切换调色板。

执行

手柄连接完成后，这里会直接把脚本发送到 ESP32 串口设备。开始前建议把 Switch 里的笔刷切到方块笔刷，整体观感通常会更美观。

请选择串口设备，连接手柄后即可直接开始绘制。
