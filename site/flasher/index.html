Friend Maker Web Flasher

打开网页，直接给 ESP32 刷固件

这个网站只做一件事：给主线 ESP32-WROOM-32 / ESP-32S 刷入 Friend Maker 固件。 不需要用户本地安装 Node、PlatformIO 或其他开发环境。

Chrome / Edge 桌面端 GitHub Pages ESP32-WROOM-32 / ESP-32S

浏览器与设备要求

首期只支持主线板型和支持 Web Serial 的桌面浏览器。

检测中 正在检查浏览器能力

  • 推荐浏览器：Chrome / Edge 桌面端
  • 目标开发板：ESP32-WROOM-32 / ESP-32S
  • 准备一根可传数据的 USB 线

      一键刷入固件

      通过 manifest 驱动的浏览器刷机流程，刷完固件后就可以进入桌面版继续连接板子并绘画。

      如果刷机后串口重新枚举或临时断开，这是正常现象。下一步请下载桌面版继续使用。

      下一步

      这个网站不负责连接手柄或发送绘画命令。刷完固件后，请直接进入成熟的桌面版流程。

      下载 Friend Maker Desktop 最新版
      1. 先用本页给开发板刷入固件
      2. 下载并打开 Friend Maker Desktop 最新版
      3. 在桌面版里继续连接板子、配对手柄并开始绘画
      连接提示

      蓝牙连接手柄通常需要等待 1-2 分钟，请先耐心等待一会儿。 如果确实一直连不上，请到 README 里查找“张老师优化版”再试。

      为什么只做刷机

      当前网页端主目标是零环境刷机入口，把后续连接与绘画继续交给稳定的桌面版处理。

      • 用户不需要安装本地开发环境就能刷固件
      • 后续绘画链路直接复用 Friend Maker Desktop 最新版
      • 网页端不承担实验性串口执行和控制器调试职责
